## Tech Stack

- Vue.js 3 (Composition API)
- Vue Router
- Pinia (State Management)
- Axios (HTTP Client)
- Tailwind CSS

## Setup
```bash
npm install
npm run dev
```

## Features

- Login/Register forms with validation
- Protected routes with navigation guards
- Real-time task progress updates
- Responsive design with Tailwind CSS
